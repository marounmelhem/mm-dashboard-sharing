== Description ==

Adds the ability to share posts directly from the dashboard panel on Facebook, Twitter, Google+, this plugin works with custom posts and pages.

== Installation ==

1. Upload `mm-dashboard-sharing` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Check the sharing buttons in the default posts page as well as the custom posts pages
4. Go to the plugin settings page to customize specific options
